ZMK firmware built for split custom DIY Ergo S-1 keyboard:
https://github.com/wizarddata/Ergo-S-1 

- Built for  ZMK development branch v0.4 (08.03.2026)
- ZMK Studio keyboard editor is Enabled in firmware https://zmk.studio/download

Flashing firmware:
- turn off battery power
- insert USB from PC to the LEFT keyboard side
- double click reset button
- new drive will popup in your system
- delete old contents from this new drive
- paste/copy ergo_v1_left-nice_nano__zmk-zmk.uf2 to this drive
- wait until drive will turn off (it will dissapear)
- remove USB cable
- repeat ALL STEPS with right side but use second file ergo_v1_right-nice_nano__zmk-zmk.uf2
